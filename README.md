# RVGL Assistant


This is the repository for RVGL Assistant program.

It's build on Python 3.6 and PyQT5 with QT Designer for GUI. Used PyCharm as IDE.

`pyrcc5 gui_resources.qrc -o gui_resources_rc.py`
